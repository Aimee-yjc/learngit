git restore <file>撤销本此修改（没有进行add）
git reset HEAD <file>重置暂存区，再进行restore操作可以撤销add过的修改
git rm <file>删除文件
误删文件如何恢复：
（1）查找删除后的commit id
git reflog
git reset --hard <commit id>
（2）撤销
